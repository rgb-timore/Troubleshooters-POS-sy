This is a basic POS system build using Python with predefine products, stocks and prices.

I will assume that whomever will run this program has some basic knowledge of python and has an IDE (integrated Developer Environment). Here is a link to a free online IDE https://www.online-ide.com/ to run the program from within the browser.

A cashier may do the following:
#Add items from the preprogrammed list using number from 1 to 12

#Remove items that were added to the customers cart by entering "r" or "R" then the item number 1 at a time

#To check out enter "q" or "Q"
